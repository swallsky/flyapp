## 安装
```
# 
yarn
```

## 安装electron
```
# yarn 
yarn config set electron_mirror https://npm.taobao.org/mirrors/electron/

# npm
npm config set electron_mirror https://npm.taobao.org/mirrors/electron/

```

### 开发模式web
```
yarn start:dev
```

### 生成mac应用
```
yarn pack:mac
```

### 生成windows应用
```
yarn pack:win
```
